What you will need to get started: 



1. Download the PI Zero image from RetroPie.
2. Write image to SD card.
3. Solder header pins into Pi.
4. Assemble case, Pi, and bonnet.
5. Install scripts for bonnet.
6. Load games.
7. Enjoy
